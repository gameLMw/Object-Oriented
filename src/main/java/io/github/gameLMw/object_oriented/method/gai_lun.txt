1. 方法重载 (Overloading)
定义：在同一个类中，多个方法具有相同的方法名但参数列表不同
特点：
    方法名相同
    参数列表不同（个数、类型、顺序）
    与返回值类型无关
    与访问修饰符无关
    编译时多态

2. 方法重写 (Overriding)
定义：子类重新定义父类中已有的方法
特点：
    方法名相同
    参数列表相同
    返回值类型相同或是其子类
    访问修饰符不能比父类更严格
    运行时多态

重要注意事项:
    @Override注解：建议始终使用，编译器会检查是否正确重写
    构造方法：不能被重写，但可以被重载
    静态方法：不能被重写，只能被隐藏
    私有方法：不能被重写
    final方法：不能被重写
    协变返回类型：重写时返回值可以是父类返回值的子类

特性          重载 (Overloading)        重写 (Overriding)
发生位置        同一个类中                   父子类之间
方法名          相同                         相同
参数列表        必须不同                    必须相同
返回值类型       可以不同                必须相同或是子类
访问修饰符       可以不同                    不能更严格
异常声明        可以不同                    不能抛出更多异常
多态类型        编译时多态                   运行时多态
绑定时间        编译期                       运行期
